🚀 PASOS PARA PONER EN FUNCIONAMIENTO
════════════════════════════════════════════════════════════════════════════


PASO 1: INSTALAR DEPENDENCIAS
──────────────────────────────

Abre tu terminal (PowerShell/CMD) y ejecuta:

    cd d:\xampp\htdocs\farmacia
    npm install

Esto descargará e instalará todos los paquetes necesarios.
Esperado: Verás líneas de instalación, no debe haber errores.

✅ Cuando termina, deberías ver una carpeta node_modules/


PASO 2: CONFIGURAR LA BASE DE DATOS
────────────────────────────────────

2.1 Abre phpMyAdmin en tu navegador:
    http://localhost/phpmyadmin

2.2 Inicia sesión (usuario: root, sin contraseña)

2.3 Ve a la pestaña SQL

2.4 Abre el archivo: database/scripts/schema.sql
    (Puedes abrirlo con el Bloc de notas)

2.5 Copia TODO el contenido

2.6 Vuelve a phpMyAdmin y pega en el área de SQL

2.7 Haz clic en "Ejecutar" (botón azul en parte inferior)

✅ Deberías ver mensajes de "Se ejecutó correctamente"

Esto creará:
  • Base de datos: farmacia_db
  • 4 tablas: medicamentos, clientes, ventas, detalles_venta
  • 3 vistas útiles
  • Datos de ejemplo


PASO 3: CREAR/CONFIGURAR .env
──────────────────────────────

3.1 En la carpeta farmacia, copia .env.example a .env

    Opción 1 (PowerShell):
    Copy-Item .env.example .env

    Opción 2 (CMD):
    copy .env.example .env

3.2 Abre .env en un editor de texto

3.3 Verifica que contenga:
    ```
    DB_HOST=localhost
    DB_USER=root
    DB_PASSWORD=
    DB_NAME=farmacia_db
    DB_PORT=3306
    SERVER_PORT=3000
    NODE_ENV=development
    ```

✅ Generalmente no necesitas cambiar nada


PASO 4: INICIAR EL SERVIDOR
────────────────────────────

En la terminal (en la carpeta farmacia):

    npm start

Deberías ver:

    ╔════════════════════════════════════════════════════════════╗
    ║  Sistema de Gestión de Farmacia                            ║
    ║  Arquitectura Hexagonal con Node.js y Express              ║
    ╚════════════════════════════════════════════════════════════╝
    
    🚀 Servidor iniciado en: http://localhost:3000
    ...

✅ El servidor está corriendo


PASO 5: ACCEDER A LA APLICACIÓN
────────────────────────────────

Abre tu navegador (Chrome, Firefox, Edge) y ve a:

    http://localhost:3000

¡Verás el Dashboard del Sistema de Farmacia! 🎉

❌ SOLUCIÓN DE PROBLEMAS COMUNES
════════════════════════════════════════════════════════════════════════════

PROBLEMA: "Cannot find module"
SOLUCIÓN: npm install

PROBLEMA: "ECONNREFUSED" (conexión rechazada a BD)
CAUSAS Y SOLUCIONES:
  • MySQL no está corriendo: Abre XAMPP y inicia MySQL
  • BD no existe: Vuelve a ejecutar schema.sql
  • Credenciales mal: Verifica .env

PROBLEMA: "Puerto 3000 está en uso"
SOLUCIÓN: 
  • Abre .env
  • Cambia SERVER_PORT=3000 a SERVER_PORT=3001
  • Reinicia: npm start
  • Accede a: http://localhost:3001

PROBLEMA: "Se carga pero sin datos"
SOLUCIÓN:
  • Abre navegador en http://localhost:3000/api/medicamentos
  • Deberías ver JSON con medicamentos
  • Si está vacío, vuelve a ejecutar schema.sql

PROBLEMA: "Modal no se abre" o "Botones no responden"
SOLUCIÓN:
  • Limpia caché del navegador (Ctrl+Shift+Supr)
  • Abre en pestaña incógnita
  • Intenta otro navegador
