🚀 CÓMO INICIAR
════════════════════════════════════════════════════════════════════════════

PASO 1: INSTALAR DEPENDENCIAS
───────────────────────────────
$ cd d:\xampp\htdocs\farmacia
$ npm install

PASO 2: CONFIGURAR BASE DE DATOS
────────────────────────────────
1. Abrir: http://localhost/phpmyadmin
2. Ir a pestaña SQL
3. Abrir archivo: database/scripts/schema.sql
4. Copiar todo el contenido
5. Ejecutar en phpMyAdmin

PASO 3: INICIAR SERVIDOR
───────────────────────
$ npm start

O modo desarrollo:
$ npm run dev

PASO 4: ACCEDER A LA APLICACIÓN
───────────────────────────────
http://localhost:3000


💡 EJEMPLO DE USO
════════════════════════════════════════════════════════════════════════════

CREAR MEDICAMENTO (vía API):
────────────────────────────
curl -X POST http://localhost:3000/api/medicamentos \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Ibuprofeno 400mg",
    "descripcion": "Antiinflamatorio",
    "precio": 7.00,
    "cantidad": 50
  }'

REGISTRAR CLIENTE:
──────────────────
POST /api/clientes
{
  "nombre": "Pedro",
  "apellido": "López",
  "dni": "87654321",
  "email": "pedro@email.com",
  "telefono": "999666777"
}

PROCESAR VENTA:
───────────────
POST /api/ventas
{
  "clienteId": 1,
  "detalles": [
    { "medicamentoId": 1, "cantidad": 2 },
    { "medicamentoId": 3, "cantidad": 1 }
  ]
}

RESULTADO:
{
  "exito": true,
  "datos": {
    "venta": { "id": 5, "clienteId": 1, "total": 45.50 },
    "totalFinal": 40.95,
    "descuentoAplicado": 4.55,
    "puntosGanados": 4,
    "puntosActuales": 54
  }
}
